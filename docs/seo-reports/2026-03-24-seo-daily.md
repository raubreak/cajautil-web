**Estado:** ❌ No disponible

El script `scripts/ga4-audit.ts` fue compilado y ejecutado exitosamente en cuanto a código, pero falló al intentar conectar con la API de Google Analytics Data:

```
Error consultando Analytics API: Error: 14 UNAVAILABLE:
Name resolution failed for target dns:analyticsdata.googleapis.com:443
```

**Causa:** El sandbox de ejecución tiene una proxy SOCKS5 configurada (`socks5h://`), pero la librería gRPC (`@grpc/grpc-js`) no soporta el esquema `socks5h:`. La resolución DNS externa está bloqueada para conexiones directas.

**Errores adicionales detectados:**
- `npx tsx` falla con `EPERM: operation not permitted` al crear pipes IPC en `/tmp/tsx-*/`
- El paquete `esbuild` instalado es para `darwin-x64` (macOS) pero el entorno es Linux — necesita `@esbuild/linux-x64`
- El registro npm está bloqueado (`403 Forbidden`) para instalar `@esbuild/linux-x64`

**Workaround aplicado:** Compilación manual con `tsc` + `NODE_PATH` para ejecutar el JS. Esto bypasea el problema de esbuild pero no soluciona el acceso de red.
